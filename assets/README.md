# Assets Directory

This directory contains visualization assets and plots generated by the linear regression implementation.

## Contents

- `ols_fit.png` - Example of OLS regression fit
- `residual_plot.png` - Residual analysis plots
- `gradient_descent_contour.gif` - Animated gradient descent optimization
- `cost_surface.png` - 3D cost function surface
- `bias_variance_tradeoff.png` - Bias-variance tradeoff illustration
- `multicollinearity_heatmap.png` - Feature correlation heatmap

## Generating Assets

Most assets are generated automatically when running the notebooks or using the visualization functions. To regenerate all assets:

```python
# Run the example script
python examples/generate_assets.py
```

Or use individual visualization functions:

```python
from src.visualization import plot_regression_line, plot_residuals
from src.linear_regression import LinearRegression
import numpy as np

# Generate sample data
X = np.random.randn(100, 1)
y = 2 * X.flatten() + 1 + 0.1 * np.random.randn(100)

# Fit model
model = LinearRegression()
model.fit(X, y)

# Create and save plots
fig = plot_regression_line(X, y, model)
fig.savefig('assets/ols_fit.png', dpi=300, bbox_inches='tight')
```
